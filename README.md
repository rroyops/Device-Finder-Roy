# 📱 FindMyPhone Wi-Fi

> Never waste time searching for your phone around the house again.

**FindMyPhone Wi-Fi** is a smart Android application that lets you locate your misplaced phone instantly—as long as both devices are connected to the same Wi-Fi network.

Instead of calling your own number or asking someone else to ring your phone, simply open the app on another paired device and trigger a loud alarm, flashlight, and vibration remotely.

---

## ✨ Features

🔍 **Automatic Device Discovery**

* Detect nearby devices running the app on the same Wi-Fi network.
* Real-time online/offline status.

📲 **Secure Device Pairing**

* One-time approval system.
* Encrypted communication.
* Trusted device management.

🔔 **Remote Phone Ringing**

* Ring your phone remotely.
* Loud customizable alarm.
* Continuous ringing until stopped.

💡 **Flashlight & Vibration**

* Blink the flashlight.
* Continuous vibration.
* Visual alert when the phone is hidden.

📊 **Live Device Information**

* Battery percentage
* Charging status
* Wi-Fi connection
* Last active time

🎨 **Modern UI**

* Material Design 3
* Dark & Light Mode
* Smooth animations
* Responsive layouts

🔒 **Privacy First**

* Works only with paired devices.
* No public server required for local discovery.
* Secure local network communication.

## 📦 Installation

1. Clone the repository.
2. Open it in Android Studio or VS Code.
3. Run `flutter pub get`.
4. Connect two Android devices to the same Wi-Fi network.
5. Build and run the application.
